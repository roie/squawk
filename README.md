# ✈️ squawk

Translate airline operations messages into plain, readable English.

Ground staff receive cryptic messages full of abbreviations like `WCHR`, `PAX`, `7M8`, `Conx`. Squawk makes them human-readable and shareable.

## Demo

[squawk-roie.vercel.app](https://squawk-roie.vercel.app)

## Features

- Paste any airline ops message
- Get a clean visual card
- Copy to clipboard
- Save as PNG
